# Soundfonts

Piano samples used for the melody player (`src/components/MelodyPlayer.tsx`). abcjs requests one
mp3 per note (`acoustic_grand_piano-mp3/<note>.mp3`, A0–C8).

We'd rather serve it locally instead of relying on abcjs's default host.

Gathered from [paulrosen/midi-js-soundfonts](https://github.com/paulrosen/midi-js-soundfonts)
(`FluidR3_GM/acoustic_grand_piano-mp3/`) — FluidR3_GM by Frank Wen, MIT license.
